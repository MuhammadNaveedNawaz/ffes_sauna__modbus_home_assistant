# FFES Sauna Modbus Integration

Control your FFES sauna controller directly from Home Assistant using Modbus TCP.

## Features

- Complete temperature control with thermostat entity
- Multiple sauna profiles (Infrared, Dry, Wet, Steam Bath, etc.)
- Session timer management
- Humidity and ventilation control
- Aromatherapy control
- Real-time error monitoring
- CPIR infrared group controls

## Quick Start

1. Install via HACS
2. Restart Home Assistant
3. Go to Settings â†’ Devices & Services
4. Click "+ Add Integration"
5. Search for "FFES Sauna"
6. Enter your controller IP address

## Requirements

- FFES Sauna Controller with Modbus TCP support
- Controller connected to your local network
- Firmware version 1.21/6.21 or newer

For full documentation, automation examples, and troubleshooting, see the [GitHub repository](https://github.com/LeszekWroblowski/ffes_sauna__modbus_home_assistant).
